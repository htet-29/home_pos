# HOME POS

This is my first ever big project for my portfolio. The main purpose of this project is to test my web knowledge, security best practice and golang use cases with minimal setup. No framework, less library as possible.

## Project Use Cases

The name 'HOME POS' explains everything. It will have CRUD operation for daily stock in/out and daily sales, report generation and user profile.

## Dependencies

- GO version 1.25.5

## Installing GO

`sudo make setup`

## Upgrade GO

`sudo make upgrade-go`
